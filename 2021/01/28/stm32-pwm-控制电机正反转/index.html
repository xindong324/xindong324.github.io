| TIM x | CH x | io pin
|:-:|:-:|:-:|
| TIM2 | CH2 | PA1 PB3 |
| TIM2 | CH3 | PA2 PB10 |
| TIM2 | CH4 | PA3 PB11 |
| TIM3 | CH1 | PC6 PB4 PA6 |
| TIM3 | CH2 | PC7 PB5 PA7 |
| TIM3 | CH3 | PC8 PB0 |
| TIM3 | CH4 | PC9 PB1 |
| TIM5 | CH1 | PA0 |
| TIM5 | CH2 | PA1 |
| TIM5 | CH3 | PA2 |
| TIM5 | CH4 | PA3 |
| TIM4 | CH1 | PB6 |
| TIM4 | CH2 | PB7 |
| TIM4 | CH3 | PB8 |
| TIM4 | CH4 | PB9 |
| TIM10 | CH1 | PB8 |
| TIM11 | CH1 | PB9 |
| TIM12 | CH1 | PB14 |
| TIM12 | CH2 | PB15 |
| TIM13 | CH1 | PA6 |
| TIM14 | CH1 | PA7 |
